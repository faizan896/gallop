# gallop

A personal investment terminal — track crypto, stocks and your overall portfolio in one place, with live prices, a watchlist, market data and cross-device sync.

**Live:** https://gallop-phi.vercel.app

## Features
- Portfolio tracking for crypto and stocks — quantities, average buy price, P&L
- Live prices from Binance, Pyth and Finnhub
- Net-worth chart, allocation breakdown and ranked holdings
- Markets page — crypto Fear & Greed, top movers and market news
- Research — look up any ticker's fundamentals
- Login + cross-device sync (Firebase); installable as a PWA
- Guest mode — opens straight into a sample portfolio, no sign-up needed

## Built with
Vanilla JavaScript · HTML · CSS (Tailwind) · Chart.js · Firebase (Auth + Firestore)

Data sources: Binance, Pyth, CoinGecko, Finnhub
